# tkinter_calculator
A simple and interactive desktop calculator built using Python’s Tkinter library. It provides a clean graphical user interface for performing basic arithmetic operations like addition, subtraction, multiplication, and division. The project focuses on understanding GUI development in Python, event handling, and building functional desktop applications with a user-friendly design.

🔹 Features
🧮 Performs basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
🖥️ Simple and clean graphical user interface using Tkinter
⌨️ Supports button-based input for easy interaction
⚡ Real-time calculation and instant results display
🧹 Clear and reset functionality to reset calculations
📱 Lightweight and runs smoothly as a desktop application
🎯 Beginner-friendly project demonstrating Python GUI development
